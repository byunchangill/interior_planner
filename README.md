# HomeStyler (interior_planner)

AI 맞춤형 인테리어 추천 웹앱. Spring Boot(백엔드) + React(프론트엔드)로 구현합니다.

## 구조

- `backend/` — Spring Boot API
- `frontend/` — React 웹앱
- `docs/` — 기획 문서 (IA, SRS, 기능 명세서), 화면 디자인(`_N/`), 디자인 시스템
- `_workspace/` — 개발 진행 상황(progress.md)

## 실행

```bash
# backend
cd backend
./gradlew bootRun

# frontend
cd frontend
npm install
npm run dev
```
