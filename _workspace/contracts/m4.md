# M4 계약 — SAVE (저장·비교·공유·구매목록)

공통 규칙은 `api-contract-guide.md` 표준을 따른다. **공개 공유 뷰(GET /share/{token})를 제외한** 모든 엔드포인트 인증 필요, 소유자 본인만(타인 → AUTH_003).

## 범위 및 결정사항

- FR-SAVE-001(저장), FR-SAVE-002(비교), FR-SAVE-003/004(공유 링크·관리), FR-SAVE-006(구매목록 내보내기)
- 화면: SAVE-001 보관함(_6), SAVE-002 비교(전용 폴더 없음 — _6 카드 확장), SAVE-003 공유(_13)
- **결정 1 (저장 모델)**: 별도 SavedRecommendation 엔티티 없이 M3 `Recommendation`에 `saved`(boolean)·`selected`(boolean) 플래그 추가. 보관함은 `saved=true` 목록. M3 소유권 헬퍼 재사용.
- **결정 2 (공유 웹뷰)**: 공개 링크 `/share/{token}`는 **비인증** 접근 → SecurityConfig에 `/api/v1/share/**` permitAll 추가 필요. 추측 불가 토큰(최소 128bit 난수, URL-safe). 보기 전용.
- **결정 3 (공유 링크 저장)**: `ShareLink` 엔티티(token, recommendationId, expiresAt nullable, includeOriginalPhotos, revoked). 만료/회수 시 공개 뷰는 SHARE_001(410).
- **결정 4 (구매목록)**: GET /recommendations/{id}/shopping-list — M3 추천안의 items·공간 치수·measureBeforeBuy를 재구성(신규 저장 없음). FE가 텍스트 복사/공유.
- ID는 Long. 공유 token은 문자열.

## 엔드포인트

### POST /recommendations/{recommendationId}/save

- **기능**: FR-SAVE-001 보관함 저장
- **응답 200**: `{ "success": true, "data": { "recommendationId": 200, "saved": true } }`
- **에러**: RES_001, AUTH_003

### DELETE /recommendations/{recommendationId}/save

- **기능**: 저장 해제
- **응답 200**: `{ "success": true, "data": { "recommendationId": 200, "saved": false } }`
- **에러**: RES_001, AUTH_003

### GET /saved

- **기능**: FR-SAVE-001 보관함 목록 (SAVE-001)
- **응답 200**:
  ```json
  { "success": true, "data": { "items": [
    { "recommendationId": 200, "spaceId": 1, "spaceName": "거실", "style": "NORDIC",
      "conceptTitle": "따뜻한 북유럽 거실", "thumbnailUrl": "/files/placeholder_nordic.png",
      "budgetTotal": 3450000, "fitScoreTotal": 92, "selected": true, "savedAt": "2026-07-20T12:00:00Z" }
  ] } }
  ```
  - 빈 보관함 `items: []`
- **에러**: 없음

### PUT /recommendations/{recommendationId}/select

- **기능**: FR-SAVE-002 "이 안으로 결정" — 같은 공간 내 대표 지정. 같은 spaceId의 다른 추천안 `selected`는 자동 해제(단일 대표)
- **응답 200**: `{ "success": true, "data": { "recommendationId": 200, "selected": true } }`
- **에러**: RES_001, AUTH_003

### POST /recommendations/compare

- **기능**: FR-SAVE-002 비교 (SAVE-002)
- **요청**: `{ "recommendationIds": [200, 201], "compareKey": "STYLE" }`
  - recommendationIds 2~3개(미달/초과 → VALID_003), 모두 본인 소유
- **응답 200**:
  ```json
  { "success": true, "data": {
    "sameSpace": true,
    "columns": [
      { "recommendationId": 200, "style": "NORDIC", "conceptTitle": "따뜻한 북유럽 거실",
        "thumbnailUrl": "/files/placeholder_nordic.png", "budgetTotal": 3450000, "fitScoreTotal": 92,
        "materialSummary": { "wallpaper": "#F5F0E8", "flooring": "#C9A227" },
        "keywords": ["화이트","우드","미니멀"] }
    ] } }
  ```
  - `sameSpace: false`면 FE가 VALID_004 안내 배너 표시(차단 아님, 계속 진행) — 서로 다른 공간이어도 200 반환하되 `sameSpace: false`
- **에러**: VALID_003(개수), AUTH_003(타인 포함), RES_001(없는 ID)

### POST /recommendations/{recommendationId}/share-links

- **기능**: FR-SAVE-003 공유 링크 생성 (SAVE-003)
- **요청**: `{ "expiresIn": "D30", "includeOriginalPhotos": false }`
  - expiresIn: D7 | D30 | NONE (기본 D30)
- **응답 200**:
  ```json
  { "success": true, "data": { "linkId": 500, "token": "a1b2c3...",
    "shareUrl": "/share/a1b2c3...", "expiresAt": "2026-08-19T12:00:00Z", "includeOriginalPhotos": false } }
  ```
  - expiresIn=NONE이면 `expiresAt: null`
- **에러**: RES_001, AUTH_003, LIMIT_002(추천안당 최대 5개)

### GET /recommendations/{recommendationId}/share-links

- **기능**: FR-SAVE-004 링크 목록(관리 UI)
- **응답 200**:
  ```json
  { "success": true, "data": { "items": [
    { "linkId": 500, "shareUrl": "/share/a1b2c3...", "expiresAt": "2026-08-19T12:00:00Z", "revoked": false, "createdAt": "2026-07-20T12:00:00Z" }
  ] } }
  ```
- **에러**: RES_001, AUTH_003

### DELETE /share-links/{linkId}

- **기능**: FR-SAVE-004 링크 회수(만료 처리)
- **응답 200**: `{ "success": true, "data": { "linkId": 500, "revoked": true } }`
- **에러**: RES_001, AUTH_003

### GET /share/{token}  (공개 — 비인증)

- **기능**: FR-SAVE-003 공개 보기 전용 웹뷰 데이터
- **인증**: **불필요** (SecurityConfig permitAll)
- **응답 200**:
  ```json
  { "success": true, "data": {
    "style": "NORDIC", "conceptTitle": "따뜻한 북유럽 거실", "conceptDescription": "...",
    "keywords": ["화이트","우드"], "budgetTotal": 3450000, "fitScoreTotal": 92,
    "materials": { "wallpaper": { "color": "#F5F0E8", "material": "실크 벽지" }, ... },
    "items": [ { "brand": "A사", "name": "3인용 패브릭 소파", "widthMm": 2180, "depthMm": 920, "heightMm": 810, "price": 1290000, "position": "거실 남측 벽" } ],
    "afterImageUrl": "/files/placeholder_nordic_after.png",
    "originalPhotos": [],
    "disclaimers": ["본 추천은 AI 분석 결과로 실제와 다를 수 있습니다."] } }
  ```
  - `includeOriginalPhotos=true`로 생성된 링크만 `originalPhotos`에 M2 원본 사진 URL 포함, 아니면 `[]`
  - 보기 전용(수정 필드·구매링크 등 민감정보 제외 가능)
- **에러**: SHARE_001(만료/회수된 링크 → HTTP 410)

### GET /recommendations/{recommendationId}/shopping-list

- **기능**: FR-SAVE-006 구매 목록 내보내기 (SAVE-001/003)
- **요청 쿼리**: `?budgetPlanRange=R300_500` (선택, 미지정 시 설정 예산 구간)
- **응답 200**:
  ```json
  { "success": true, "data": {
    "spaceSummary": { "widthM": 4.5, "depthM": 3.6, "heightM": 2.3 },
    "measureBeforeBuy": ["남측 벽 길이", "현관 폭"],
    "items": [
      { "brand": "A사", "name": "3인용 패브릭 소파", "widthMm": 2180, "depthMm": 920, "heightMm": 810,
        "price": 1290000, "purchaseUrl": "https://example.com/p/301" } ],
    "totalPrice": 3450000 } }
  ```
- **에러**: RES_001, AUTH_003

## Enums

- CompareKey: STYLE | BUDGET | LAYOUT | MATERIALS (기본 STYLE)
- ShareExpiry: D7 | D30 | NONE
- (StyleType/BudgetRange 등은 M1/M3 재사용)

## 에러 코드

| 코드 | 의미 |
|------|------|
| AUTH_003 | 접근 권한 없음 |
| RES_001 | 리소스 없음 |
| VALID_003 | 비교 개수 미달/초과(2~3개 아님) |
| VALID_004 | 서로 다른 공간 추천안 비교(경고, 차단 아님 — 응답은 200 + sameSpace:false로 대체) |
| SHARE_001 | 만료/회수된 공유 링크 (공개 뷰, HTTP 410) |
| LIMIT_002 | 공유 링크 한도 초과(추천안당 5개) |

## SecurityConfig 변경

- `/api/v1/share/**` permitAll 추가 (공개 웹뷰). 나머지 M4 엔드포인트는 기존 `anyRequest().authenticated()` 유지.
