# M0 계약 — 스캐폴딩

공통 규칙은 `api-contract-guide.md` 표준을 따른다 (Base URL `/api/v1`, ApiResponse 포맷, camelCase).

## 환경 (확인됨)

- Java 21 (Temurin), Node 24 / npm 11, Windows 11 + Git Bash
- BE 포트 **8080**, FE dev 포트 **5173**. FE dev proxy: `/api` → `http://localhost:8080`

## 엔드포인트

### GET /api/v1/health

- **기능**: 스캐폴딩 연동 확인용 헬스체크
- **인증**: 불필요
- **응답 200**:
  ```json
  { "success": true, "data": { "status": "UP", "service": "homestyler-api", "timestamp": "2026-07-19T12:00:00Z" } }
  ```
- **에러**: 없음

## 백엔드 산출 범위 (backend/)

- Spring Boot 4.1.0 + Java 21 toolchain + Gradle 9.5 wrapper (Initializr가 3.x 미제공 → 4.1.0 승인됨. Jackson 3 / Security 7 기준)
- `common/ApiResponse`, `common/exception/`(ErrorCode enum + ApiException + GlobalExceptionHandler), CORS 설정(5173 허용)
- Security: `/api/v1/health`, `/api/v1/auth/**`, `/files/**` permitAll, 나머지 인증 필요(지금은 스텁), CSRF off
- 프로파일: 기본 dev(H2, ddl-auto update). HealthController 구현

## 프론트엔드 산출 범위 (frontend/)

- Vite + React 18 + TypeScript + Tailwind
- `docs/homestyler_design_system/DESIGN.md` frontmatter의 colors/typography/rounded를 tailwind config로 이식 (클래스명 = 토큰명: `bg-primary`, `text-on-surface` 등). Google Fonts(Plus Jakarta Sans, Inter) + Material Symbols는 index.html에서 로드
- react-router 셸: `/`(홈 placeholder), `/spaces`, `/reco`, `/saved`, `/my` + 하단 탭 네비게이션(BottomNav 컴포넌트, 디자인 시스템 스타일)
- `api/client.ts`: axios 인스턴스(`baseURL: '/api/v1'`) + 토큰 인터셉터 스텁(localStorage accessToken 첨부, 401 시 refresh 재시도 자리) + `data` unwrap 헬퍼
- 홈 화면에 /health 호출 결과 표시(연동 검증용, 추후 교체)
