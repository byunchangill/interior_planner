# 화면 폴더 ↔ 화면ID 매핑

**주의**: `docs/_34.md`의 SCREEN_N 번호는 이 폴더 번호와 다른 체계다. 이 표가 유일한 기준이다.

## 전용 디자인이 없는 화면 (RECO-005, SAVE-002)

새로 디자인하지 말고: (1) 인접 화면(`_4`, `_6`)의 카드·섹션·타이포 패턴을 재사용하고 (2) `docs/homestyler_design_system/DESIGN.md`의 컴포넌트 규칙(카드 24px 라운딩, AI 요소 글래스모피즘, 버튼 48px 높이)을 따른다.
